Website Committee

The Website Committee takes care of the ACM Website. They would update the website with new pictures, information, and features. This website in made using the SvelteKit package, the tailwindcss library, and additional tools making it stream lined and easy for programmers to jump right in. In the Website Committee you'll learn skills like team work, proper website design, user interactivity, and the importance of code review.

Rules

  • There needs to be one project leader that is in charge of organizing and reviewing any changes to the site.
  • A basic level of how GitHub works before writing any code.
  • Branches should be based on features.
  • Keep to the color scheme.
  • Reupload the old site only if the current site has issues.
  • Everyone has to have a GitHub account.

Recommendations

  • Meet weekly or biweekly to discuss website plans.
  • Understanding of HTML, JavaScript, and CSS.
  • Anyone new to web development, SvelteKit, or Tailwindcss should only make small changes like updating or adding photos.
  • Have common a goal and outline for a feature to be completed.
